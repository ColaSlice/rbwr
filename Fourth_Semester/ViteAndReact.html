Vite + React

Det har alrig været nemmere. Jeg behøver ikke at sige mere. Ok, jeg burden nok forklare lidt mere i dybden. Vite er et build tool som forenkler din tool-chain. Færre filer og færre dependencies. Det gør det ikke kun nemmere at få overblik over ens project, men der er også færre dependencies at tænke på. Hvis jeg køre en normal npm create-react-app app, så ville jeg ende ud med 40.000 filer i projeket. Det er circa 300-400 MB. Men hvis jeg køre npm create vite, og vælger React og JavaScript, så ender jeg ud med circa 2,494 filer. Det er circa 40 MB. Der er aligevel en lille forskel der. Med Vite vil den by default bruge .jsx filer istedet for .js filer, når man laver et React projekt.

Når man skal starte et nyt React project med Vite, skriver man npm create vite, og Vite vil derefter starte en promt, hvor man indtaster information til projektet. Det er projekt navn, framework og sprog, som man vælger. Neden under er et billede, med hele promten. Lige en hurtig rettelse, hust at køre npm install, før at du kan køre projektet.
